    Class VGSShow

    Reveal orchestrator for VGS Show React Native components.

    VGSShow owns vault routing, request building, custom headers, reveal dispatch, and subscription fan-out. Subscribed components receive decoded content internally after request() succeeds.

    Remarks

    Public callbacks and refs intentionally expose only status, errors, actions, and logical booleans.
